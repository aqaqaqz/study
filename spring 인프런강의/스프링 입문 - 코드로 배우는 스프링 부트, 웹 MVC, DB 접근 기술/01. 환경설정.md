# 환경설정
- 프로젝트 생성
    - https://start.spring.io
- 버전
    - Spring Boot [3.2.4]
        - 3.x이상 버전은 jdk 17 이상을 사용하여야 함
    - jdk [17] 
- Dependencies
    - Spring Web WEB
    - Thymeleaf (템플릿 엔진)
- 빌드 [Gradle]
    - window : gradlew.bat build
    - linux : gradlew build
    - /build/libs에 실행파일 생성
- 스프링 가이드문서에 익숙해지기
    - welcome page : https://docs.spring.io/spring-boot/docs/current/reference/html/web.html#web.servlet.spring-mvc.welcome-page
    - 필요한 정보를 찾을 수 있어야 한다
